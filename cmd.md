# Подсказка по командной строке

## Как сменить директорию

Windows:
```sh
cd C:\Path\to\folder
```
Linux\Mac OS
```sh
cd /Path/to/folder
```

## Работа с git

Инициализация проекта
```sh
git init
```
Отслеживание изменений
```sh
git status
```

Добавить файл к отслеживанию
```sh
git add 'file'
```

Добавить сообщение(коммит) 
```sh
git commit -m 'text'
```
